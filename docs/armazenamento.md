# Armazenamento
  
Os resultados do RSI são armazenados automaticamente no banco `SQLite` definido (padrão: `rsi.db`).
  
Tabela: `rsi`
  
| Coluna | Descrição           |
|--------|---------------------|
| RSI    | Valor do indicador  |
| close  | Preço de fechamento |
| time   | Timestamp            |
